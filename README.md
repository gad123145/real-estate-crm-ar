# نظام إدارة العملاء العقاريين

تطبيق عربي لإدارة عملاء التسويق العقاري يعمل بحسابات مستخدمين وقاعدة بيانات Supabase. يحفظ بيانات الملاك والعقارات، العملاء الطالبين للشراء أو الإيجار، والمواعيد والمعاينات مع كشف تعارض التوقيتات.

## المميزات

- قسم للملاك والعقارات: نوع العقار، بيع أو إيجار، المنطقة، السعر، المساحة، التشطيب، الحالة، وملاحظات المتابعة.
- قسم للعملاء الطالبين: نوع الطلب، الميزانية، المناطق المطلوبة، درجة الاستعجال، تفاصيل الطلب، والحالة.
- قسم للمواعيد والمعاينات: ربط الموعد بمالك أو طالب، التاريخ، الوقت، المدة، المكان، والحالة.
- تاريخ إضافة تلقائي لكل سجل.
- تعديل وحذف لكل مالك، طالب، أو موعد.
- تنبيهات للمواعيد المتأخرة وتعارض المواعيد.
- تسجيل دخول وإنشاء حساب عبر Supabase Auth.
- حفظ سحابي في Supabase مع سياسات أمان تفصل بيانات كل مستخدم.

## التشغيل

```bash
npm install
npm run dev
```

انسخ ملف `.env.example` إلى `.env.local` وضع بيانات Supabase:

```env
VITE_SUPABASE_URL=your-supabase-project-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
```

ثم افتح الرابط الذي يظهر في الطرفية، غالبًا:

```text
http://127.0.0.1:5173/
```

## الفحص

```bash
npm run lint
npm run build
```

## النشر

يمكن نشر النسخة المبنية على GitHub Pages من فرع `gh-pages`. عند النشر تحت مسار مستودع GitHub Pages استخدم:

```powershell
$env:VITE_BASE_PATH='/real-estate-crm-ar/'
npm run build
```
